# Parser Engine

The parser engine converts supported document formats into normalized text and structured parsing results.

## Supported Formats

- TXT
- Markdown
- CSV
- XLSX
- HTML
- EPUB
- JSON
- XML

## Features

- Automatic parser selection through the factory
- Encoding detection and UTF-8 normalization
- Text cleaning and whitespace normalization
- Paragraph and heading detection
- Table extraction for CSV content
- Character, word, and token counting
- Parsing statistics and structured results
